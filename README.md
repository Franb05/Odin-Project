# Odin-Project
Pagina de Inicio The Odin Project

Landing Page - The Odin Project
Descripción

Este proyecto consiste en una Landing Page desarrollada como parte del curso Foundations de The Odin Project. El objetivo fue recrear un diseño dado mientras se practicaban los conceptos fundamentales del desarrollo web.

Tecnologías utilizadas
HTML5
CSS3
Flexbox

Características
Estructura HTML semántica.
Diseño basado en Flexbox.
Encabezado con barra de navegación.
Sección principal (Hero) con imagen y botón de acción.
Sección de productos.
Sección de cita utilizando aside.
Sección de llamado a la acción.
Pie de página.

Conceptos trabajados
HTML
Uso de elementos semánticos:
header
nav
main
section
aside
footer

Organización correcta del contenido.
Uso de clases en lugar de IDs innecesarios.
Accesibilidad mediante atributos alt en las imágenes.

CSS
Colores y tipografía.
Espaciado y alineación.
Estilos para botones y efectos hover.
Bordes y fondos.
Incorporación de Google Fonts.
Organización y mantenimiento del archivo CSS.
Flexbox
Distribución horizontal y vertical de elementos.
Uso de justify-content.
Uso de align-items.
Separación entre elementos con gap.
Centrado de secciones mediante margin: 0 auto.
